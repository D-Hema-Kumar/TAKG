# TAKG (Time Aware Knowledge Graph)

To use the knowledge graph:

step 1: Download the takg.ttl file to your local <br>
step 2: Open anaconda CLI and execute <pip install rdflib-endpoint> <br>
step 3: Change to the downloaded directory of the takg.ttl file <br>
step 4: execute <rdflib-endpoint serve *.ttl> <br>
step 5: Access the SPARQL interface <http://0.0.0.0:8000> for macOS <br>
